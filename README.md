## FOR SHELLEMU INFO:
ShellEmu is a shell emulator with:
- Virtual Users,
- other OS shells on other OSes, ex: CMD on Android/Linux, Powershell on Android/Linux, and Linux on Windows/Android (VIRTUAL USERS IN BETA),
- Virtual Storage,
- etc!
